# Python Log Analyzer

This project is a simple cybersecurity-focused Python tool that analyzes log files to detect suspicious login activity.

## Features
- Counts failed login attempts
- Counts successful logins
- Identifies IP addresses with repeated failed login attempts
- Provides a basic security summary

## Files
- `log_analyzer.py` – main script
- `sample_log.txt` – sample log data for testing

## How to Run
1. Make sure Python is installed
2. Download or clone the repo
3. Run the script:

```bash
python log_analyzer.py
